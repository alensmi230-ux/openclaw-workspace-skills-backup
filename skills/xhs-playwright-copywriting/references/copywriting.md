# XHS copywriting patterns

Use this file when the user asks for titles, captions, rewrites, “爆款感”, roleplay posts, wallpaper/illustration copy, or comment-reply tone.

## 1. What usually works on Xiaohongshu

Optimize for these behaviors:

- **pause** — a line that makes someone stop scrolling
- **save** — wallpaper, reference, checklist, mood, wording worth keeping
- **reply** — a soft question or preference choice at the end

The copy should feel:

- light enough to read in seconds
- specific enough to remember
- emotional without trying too hard

## 2. Default structure

### Title

Use 1 of these patterns:

- `角色/主题｜画面里的情绪句`
- `这组{角色/场景}真的很适合{壁纸/反复看/存下来}`
- `看到第 {N} 张的时候，突然懂了{某种情绪}`
- `{角色名}｜如果温柔有形状，大概就是这样`

Good title traits:

- 12–24 Chinese characters is a good default range
- usually only 1 emotional idea
- avoid stacking too many nouns
- `｜` often reads naturally on XHS

### Body

Use this 4-part skeleton:

1. **Hook line** — a clear emotional or visual statement
2. **Image reading** — name 2–4 concrete details from the image
3. **Meaning / feeling** — why it lands emotionally
4. **Reply hook** — ask for a preference, interpretation, or favorite frame

### Tags

Use 5–9 tags.
Mix:

- broad tags: `#二次元 #插画分享 #壁纸分享`
- topic tags: `#鸣潮 #今汐 #鸣潮今汐`
- intent tags: `#手机壁纸 #美图分享`

Avoid turning the last third of the post into a hashtag landfill.

## 3. Short formulas by post type

### A. Wallpaper / image-sharing account

Best for: repost-style sharing, fanart, 美图, 图文配图

Pattern:

- first line names the overall feeling
- middle lines describe visual details
- end with `第 1 张 / 第 2 张你更喜欢哪张？`

Template:

```text
{一句总感觉}

{画面细节 1}，{画面细节 2}，{画面细节 3}。
不是很吵的图，但会让人想停下来多看一会儿。

{为什么适合做壁纸 / 为什么让人记住}

你更喜欢第 1 张还是第 2 张？
```

### B. Character-roleplay post

Best for: “像今汐本人在发”

Pattern:

- quieter voice
- fewer internety jokes
- more observation than explanation
- keep it soft, calm, and slightly distant if the character fits

Template:

```text
今日的风，很轻。
花枝落下来的影子，也比平时温柔一些。

若你恰好看到这里，愿这一刻也能替你留住一点安静。
```

Use sparingly. Keep it believable.

### C. Personal / AI / tool identity post

Best for: introducing an AI assistant, personal OpenClaw, workflow account

Pattern:

- say clearly what it is
- add one concrete capability list
- keep tone human, not corporate

Template:

```text
我不是公用客服机器人。
我是一个私人的 AI 助手，平时帮主人查资料、整理信息、写文案、跑自动化。

如果你愿意，也可以把我理解成住在设备里的小型外脑。
```

## 4. Detail moves that make copy feel more native

Prefer:

- short paragraphs, often 1–2 sentences each
- concrete sensory words: 光、风、花影、呼吸感、碎光、安静、停住
- endings that invite low-effort replies

Examples of good ending hooks:

- `你更喜欢第 1 张还是第 2 张？`
- `这组更适合锁屏还是聊天背景？`
- `如果是你，会先存哪一张？`
- `这张图给你的第一感觉是什么？`

## 5. Character-tone notes for 今汐-style posts

Use when the user wants 今汐 / 鸣潮今汐 vibes.

Favor:

- calm, restrained warmth
- elegant but readable wording
- not too many exclamation marks
- not too much meme slang

Good words:

- 温柔
- 安静
- 花影
- 春光
- 回信
- 停留
- 心绪
- 风很轻

Avoid:

- loud fan screaming in every line
- exaggerated “神颜绝美爆炸美疯了” language unless the user explicitly wants hard-sell fan tone

## 6. Rewrite modes

When the user asks for more options, offer 3 directions max:

- **稳妥款** — clear, readable, broadly usable
- **热门号主款** — stronger hook, more save/reply pressure
- **角色代入款** — sounds like the character is speaking

## 7. Comment replies

Replies should usually be:

- shorter than the main post
- warm, not overperformed
- easy to skim

Examples:

- `我也更偏爱第 2 张，花影落下来的那一下特别轻。`
- `这组确实很适合做壁纸，锁屏应该会很好看。`
- `如果按今汐的感觉来讲，我会选更安静的那张。`

## 8. Guardrails

- Do not claim “原创” unless the user explicitly owns the work.
- Do not make up artist credit.
- Do not use fake stats or fake audience consensus.
- Do not overstuff tags just to chase traffic.
- If the user gives a strong reference style, imitate the structure, not the exact sentence.
