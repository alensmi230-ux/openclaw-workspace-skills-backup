# Prompting guide

## Aspect ratios

- `1:1` → square social posts, icons
- `16:9` → slides, banners, hero images
- `9:16` → stories, reels, vertical posters
- `4:3` or `3:4` → classic landscape/portrait layouts
- `21:9` → ultra-wide headers

## Sizes

- `1K` → quick drafts, low-cost iteration
- `2K` → default for most usable outputs
- `4K` → final large export when the user explicitly needs it

## Prompt structure

A strong prompt usually includes:

1. **Subject** — who or what is in the image
2. **Scene** — setting or environment
3. **Style** — photorealistic, watercolor, flat illustration, etc.
4. **Composition** — close-up, wide shot, centered, minimal, etc.
5. **Lighting / mood** — golden hour, moody neon, soft studio light
6. **Constraints** — no text, clean background, preserve pose, etc.

## Short examples

### Hero image

> futuristic AI workspace, dark navy gradient background, subtle purple glow, minimal composition, premium SaaS landing page aesthetic, no text

### Product banner

> sleek wireless headphones on a reflective surface, dramatic studio lighting, soft blue rim light, premium advertising photography, clean background

### Edit example

> keep the product shape and position unchanged, replace the background with soft beige paper texture, add natural daylight shadows, clean editorial style

### Multi-image example

> use the first image as the main subject and the second image as the environment reference, place the subject naturally into the scene, match lighting and scale

## Common failure fixes

- Too vague → add subject, style, and composition details
- Wrong scene → state the setting explicitly
- Subject drift during editing → say what must remain unchanged
- Inconsistent batch outputs → create one template first and reuse it
